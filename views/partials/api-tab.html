API Reference

GET /api/issues/{project}

Get all issues for a project. Supports filtering via query parameters.

Query Parameters:
  • open - Filter by issue status (true/false)
  • assigned_to - Filter by assignee
  • _id - Filter by issue ID

POST /api/issues/{project}

Create a new issue.

Request Body:
  • issue_title* - Title of the issue
  • issue_text* - Detailed description of the issue
  • created_by* - Name of the issue creator
  • assigned_to - Name of person assigned to the issue (optional)
  • status_text - Current status (optional)

* Required fields

PUT /api/issues/{project}

Update an existing issue.

Request Body:
  • _id* - ID of the issue to update
  • issue_title - Updated title
  • issue_text - Updated description
  • created_by - Updated creator name
  • assigned_to - Updated assignee
  • status_text - Updated status
  • open - Issue state (true/false)

* Required field, others optional

DELETE /api/issues/{project}

Delete an issue.

Request Body:
  • _id* - ID of the issue to delete

* Required field
